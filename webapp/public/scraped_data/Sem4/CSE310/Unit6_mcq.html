24 Which statement best describes the performance characteristics of adding an element to the beginning of a large ArrayList versus a large LinkedList?

Correct Answer: LinkedList is faster because it only requires updating a few node references.

Explanation:

Adding an element to the beginning of an ArrayList is an O(n) operation, where n is the number of elements.
