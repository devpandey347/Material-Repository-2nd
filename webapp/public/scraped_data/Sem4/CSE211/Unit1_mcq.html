44 Consider a computer that adds a new instruction: SWPM R1, (R2). This instruction swaps the content of register R1 with the content of the memory location whose address is in register R2. Which of the following sequence of micro-operations, using a temporary register TEMP, correctly implements the execution phase of this instruction on a single-bus architecture?

computer registers Hard
A. 1. MAR <- R2, 2. TEMP <- M[MAR], 3. MBR <- R1, 4. M[MAR] <- MBR, 5. R1 <- TEMP
B. 1. MAR <- R2, 2. MBR <- R1, 3. TEMP <- M[MAR], 4. M[MAR] <- MBR, 5. R1 <- TEMP
C. 1. TEMP <- R1, 2. MAR <- R2, 3. MBR <- M[MAR], 4. R1 <- MBR, 5. MBR <- TEMP, 6. M[MAR] <- MBR
D. 1. MAR <- R2, 2. Memory Read, 3. TEMP <- MBR, 4. MBR <- R1, 5. Memory Write, 6. R1 <- TEMP

Correct Answer: 1. MAR <- R2, 2. Memory Read, 3. TEMP <- MBR, 4. MBR <- R1, 5. Memory Write, 6. R1 <- TEMP

Explanation:

This operation requires reading from memory, writing to memory, and moving data between registers. On a single-bus architecture, data to/from memory must pass through the MBR. A temporary register (TEMP) is needed to hold the value from memory while it's being overwritten.

  1. MAR <- R2: The address from R2 is placed on the Memory Address Register.
  2. Memory Read: A memory read control signal is issued. The content of M[MAR] is fetched and placed in the Memory Buffer Register (MBR).
  3. TEMP <- MBR: The value from memory is saved into an internal CPU register TEMP.
  4. MBR <- R1: The value from R1 is placed into the MBR, preparing for the write to memory.
  5. Memory Write: A memory write control signal is issued. The content of MBR (which is the original R1 value) is written to M[MAR].
  6. R1 <- TEMP: The original memory value, saved in TEMP, is copied back to R1. This completes the swap.
    Option A is incorrect because TEMP <- M[MAR] is not a valid micro-operation; data from memory must go via MBR. Option B has the same flaw. Option D performs the operations in a less efficient order and uses M[MAR] notation ambiguously; explicit read/write signals are clearer and more accurate.

61 Which of the following correctly pairs a computer component with its primary function during the instruction fetch phase?

computer organization Hard
A. Memory Buffer Register (MBR): Receives the instruction word from the memory unit.
B. Program Counter (PC): Stores the instruction that is currently being executed.
C. Memory Address Register (MAR): Is loaded with the content of the MBR to specify the memory location.
D. Instruction Register (IR): Holds the address of the next instruction to be fetched.

Correct Answer: Memory Buffer Register (MBR): Receives the instruction word from the memory unit.

Explanation:

During the instruction fetch phase:

  1. The address of the instruction is in the Program Counter (PC).
  2. The content of the PC is copied to the Memory Address Register (MAR): MAR <- PC.
  3. The control unit sends a 'read' signal to memory.
  4. Memory places the instruction word at the address specified by the MAR onto the data bus.
  5. The Memory Buffer Register (MBR), also known as the Memory Data Register (MDR), receives this instruction word from the data bus.
  6. The content of the MBR is then copied to the Instruction Register (IR): IR <- MBR.
    Therefore, the MBR's key function in this phase is to act as a temporary holding register for the instruction coming from memory.
