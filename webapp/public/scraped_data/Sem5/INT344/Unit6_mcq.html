1 What is the primary concept behind Transfer Learning in the context of Question Answering (QA)?

A. Training a model from scratch on a small QA dataset
B. Translating questions from one language to another before answering
C. Using hard-coded rules to extract answers from text
D. Using a model pre-trained on a large corpus and fine-tuning it on a QA dataset

Correct Answer: Using a model pre-trained on a large corpus and fine-tuning it on a QA dataset

Explanation:

Transfer learning involves taking knowledge gained from solving one problem (pre-training on a large corpus) and applying it to a different but related problem (fine-tuning on specific QA data).

13 What technique does the Reformer use to approximate the attention mechanism efficiently?

A. Locality Sensitive Hashing (LSH)
B. Recurrent connections
C. Global Average Pooling
D. Convolutional layers

Correct Answer: Locality Sensitive Hashing (LSH)

Explanation:

Reformer uses Locality Sensitive Hashing (LSH) attention to group similar vectors together, allowing it to compute attention only among relevant tokens, reducing complexity from O(N^2) to O(N log N).

14 In a Reformer model, what is the purpose of Reversible Residual Layers?

A. To store activations only once, allowing recalculation during backpropagation to save memory
B. To reverse the text direction for bidirectional context
C. To increase the number of parameters without increasing size
D. To allow the model to run on CPUs only

Correct Answer: To store activations only once, allowing recalculation during backpropagation to save memory

Explanation:

Reversible layers allow the activations of the previous layer to be computed from the next layer, meaning the model doesn't need to store all intermediate activations in memory for backpropagation.

28 In the Reformer, what happens if two vectors fall into the same hash bucket during LSH?
