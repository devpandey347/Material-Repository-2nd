1 What is the primary purpose of using External Scripts in Automation 360?

A. To replace the Control Room database
B. To extend the capabilities of bots beyond native actions
C. To automatically generate documentation
D. To increase the license cost of the RPA solution

Correct Answer: To extend the capabilities of bots beyond native actions

Explanation:

External scripts allow developers to perform complex logic, calculations, or integrations that are not available as out-of-the-box actions in the Automation 360 command library.

B. Swift

11 When passing sensitive data (like passwords) to an external script, what is the most secure method?

A. Write the password to a text file and have the script read it
B. Hardcode the password in the script file

16 When using the DLL package, what is the difference between Managed and Unmanaged code?

B. Managed code requires a manager approval

49 What is the significance of the __name__ == "__main__" block in Python scripts used with Automation 360?

A. It is useful for testing the script locally but is ignored when calling specific functions via the Bot
B. It is required for the script to run
C. It renames the bot
D. It causes an error in Automation 360

Correct Answer: It is useful for testing the script locally but is ignored when calling specific functions via the Bot

Explanation:

This block executes code only if the script is run directly (standalone). When Automation 360 imports the script to call a specific function (e.g. def process():), the code inside if __name__ == "__main__": does not run, preventing side effects.
