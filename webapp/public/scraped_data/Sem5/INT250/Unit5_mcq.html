1 Which part of the internet is not indexed by standard search engines and requires specific software, configurations, or authorization to access?

A. Deep Web
B. Surface Web
C. Dark Web

A. Garlic Routing
B. Packet Switching
C. Onion Routing
D. Tomato Routing

Explanation:

Tor uses Onion Routing, where data is encapsulated in layers of encryption, peeled off one by one by relay nodes.

The .onion TLD denotes an anonymous hidden service reachable via the Tor network.
