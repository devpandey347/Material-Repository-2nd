1 Which PowerShell cmdlet is primarily used to load a module into the current session?

A. Install-Module
B. Get-Module
C. Import-Module
D. Load-Module

Correct Answer: Import-Module

Explanation:

Import-Module is the specific cmdlet used to load a module into the active session. While Get-Module lists loaded or available modules, Install-Module downloads them from a repository.

Incorrect! Try again.

B. $env:ModulePath
C. $env:Path
D. $env:PSHome

Correct Answer: $env:PSModulePath

Explanation:

$env:PSModulePath contains the list of directory locations where PowerShell searches for modules to auto-load or import.

Incorrect! Try again.

3 What is the file extension for a PowerShell Module Manifest file?

A. .xml
B. .psd1
C. .ps1
D. .psm1

Correct Answer: .psd1

Explanation:

A .psd1 file is a module manifest, which is a hash table containing data about the module, such as version, author, and dependencies. .psm1 is the script module file.
