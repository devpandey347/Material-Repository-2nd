1 Which Windows component is responsible for storing local user account passwords in a hashed format?

A. Winlogon.exe
B. pagefile.sys
C. SAM (Security Account Manager)
D. NTDS.dit

Correct Answer: SAM (Security Account Manager)

Explanation:

The Security Account Manager (SAM) database stores local user accounts and security descriptors, including password hashes, located at C:\Windows\System32\config\SAM.
