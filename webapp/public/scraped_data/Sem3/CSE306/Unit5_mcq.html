A. Link State Routing
B. Distance Vector Routing
C. Path Vector Routing

Explanation:

The Count-to-Infinity problem occurs in Distance Vector Routing when incorrect routing information propagates slowly (slow convergence) after a link failure, causing loops.

Split Horizon is a technique where a router does not advertise routes back to the neighbor from whom it learned those specific routes, preventing basic loops.

7 In the Leaky Bucket algorithm, what happens if a packet arrives when the bucket is full?

A. The bucket expands to accommodate the packet.
B. The packet is sent immediately at a higher rate.
C. The packet is discarded (dropped).
D. The packet is queued indefinitely.

Correct Answer: The packet is discarded (dropped).

Explanation:

In the Leaky Bucket algorithm, the bucket has a fixed capacity. If data arrives when the bucket is full, the incoming packets are discarded (dropped).

8 What is the primary difference between Leaky Bucket and Token Bucket algorithms?

A. Leaky Bucket enforces a constant output rate; Token Bucket allows for bursty traffic.
B. Leaky Bucket is used for routing; Token Bucket is used for error control.
C. Leaky Bucket allows bursty traffic; Token Bucket enforces a constant rate.
D. There is no functional difference.

Correct Answer: Leaky Bucket enforces a constant output rate; Token Bucket allows for bursty traffic.

Explanation:

A Leaky Bucket smooths traffic to a constant rate (like water leaking from a hole). A Token Bucket accumulates tokens, allowing a burst of data to be sent if enough tokens have accumulated.

Incorrect! Try again.

15 Which of the following describes Flooding?

A. Packets are sent based on a probability distribution.
B. A packet received on one line is sent out on all other lines.

Flooding is a routing technique where every incoming packet is sent out on every outgoing line except the one it arrived on.

B. Areas

17 What is Backpressure in the context of congestion control?

A. Node-by-node congestion propagation in the reverse direction of data flow.
B. Discarding packets at the tail of the queue.
C. Increasing the window size to push more data.
D. Prioritizing voice traffic over data traffic.

Correct Answer: Node-by-node congestion propagation in the reverse direction of data flow.

Explanation:

Backpressure is a technique where a congested node stops receiving data from its upstream neighbor, causing the upstream neighbor to eventually become congested, propagating the 'pressure' back to the source.

19 The Token Bucket algorithm requires which two parameters?

20 Which mechanism allows a router to randomly drop packets before its queue becomes full to prevent global synchronization of TCP sessions?

A. Tail Drop
B. RED (Random Early Detection)
C. Leaky Bucket
D. Priority Queuing

Correct Answer: RED (Random Early Detection)

Explanation:

RED monitors the average queue size and drops packets probabilistically before the buffer overflows, signaling congestion early to TCP sources so they reduce their window size desynchronized.

24 In the TCP Congestion Control mechanism (often discussed in network layer contexts regarding flow interaction), what happens during the 'Slow Start' phase?

25 What is the 'Poison Reverse' technique?
