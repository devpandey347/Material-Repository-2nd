1 What is the output of the following code?
str = 'Python'
print(str[1:4])

A. yth
B. tho
C. ytho
D. pyt

Correct Answer: yth

Explanation:

String slicing includes the start index (1, which is 'y') and excludes the end index (4, which is 'o'). So it prints indices 1, 2, and 3.

Incorrect! Try again.

2 Strings in Python are:

A. Dynamic arrays
B. Immutable

Once a string is created in Python, its elements cannot be changed. Any operation that modifies it actually creates a new string.
