1 What does CSS stand for?

A. Computer Style Sheets
B. Colorful Style Sheets
C. Creative Style Sheets
D. Cascading Style Sheets

Correct Answer: Cascading Style Sheets

Explanation:

CSS stands for Cascading Style Sheets. It is used to describe the presentation of a document written in HTML or XML.

A. <link>
B. <css>
C. <style>
D. <script>

Correct Answer: <style>

Explanation:

The <style> element is used to define internal CSS styles, typically placed within the <head> section of an HTML page.

Incorrect! Try again.

3 Which HTML attribute is used to define inline styles?

B. styles
C. style
D. class

Explanation:

D. <a href="mystyle.css">
