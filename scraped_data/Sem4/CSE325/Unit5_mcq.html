1 Which header file must be included to use POSIX threads in a C program?

A. <sys/thread.h>
B. <thread.h>
C. <process.h>
D. <pthread.h>

Correct Answer: <pthread.h>

Explanation:

The standard header file for the POSIX thread library is <pthread.h>.

Incorrect! Try again.

2 Which compiler flag is typically required to compile a multithreaded program using gcc on Linux?

A. -pthread
B. -thread
C. -kthread
D. -lpthread

Correct Answer: -pthread

Explanation:

The -pthread flag tells the compiler to link the pthread library and configure compilation for threads. -lpthread links the library but doesn't necessarily set preprocessor macros correctly.

Incorrect! Try again.

3 What is the return type of the function pthread_create?

A. int
B. void
C. void *
D. pthread_t

Correct Answer: int

Explanation:

pthread_create returns an integer: 0 indicates success, while a non-zero value indicates an error number.

Incorrect! Try again.

4 In the function pthread_create, what is the purpose of the fourth argument?

A. It sets the thread attributes.
B. It is the argument passed to the start_routine.
C. It stores the thread ID.
D. It is the function pointer to the thread routine.

Correct Answer: It is the argument passed to the start_routine.

Explanation:

The signature is int pthread_create(pthread_t *thread, const pthread_attr_t *attr, void *(*start_routine) (void *), void *arg). The fourth argument is passed to the start routine.

Incorrect! Try again.

9 Why is pthread_equal(id1, id2) used instead of the == operator?

B. Because pthread_t is an opaque data type and may be a structure.

10 What is a Race Condition?

A. A condition where threads race to finish execution first.
B. A mechanism to speed up thread execution.
C. A situation where a thread waits indefinitely for a resource.
D. A situation where multiple threads access shared data concurrently and the outcome depends on execution order.

Correct Answer: A situation where multiple threads access shared data concurrently and the outcome depends on execution order.

Explanation:

A race condition occurs when the system's substantive behavior is dependent on the sequence or timing of other uncontrollable events.

Incorrect! Try again.

11 Which of the following is NOT a requirement for a solution to the Critical Section Problem?

A. First-In-First-Out (FIFO) Execution
B. Bounded Waiting
C. Progress
D. Mutual Exclusion

Correct Answer: First-In-First-Out (FIFO) Execution

Explanation:

The three requirements are Mutual Exclusion, Progress, and Bounded Waiting. FIFO is a scheduling strategy, not a strict requirement for solving the CS problem (though bounded waiting implies no starvation).

Incorrect! Try again.

12 A segment of code where shared resources are accessed is called a:

A. Race condition
B. Atomic region
C. Critical section
D. Mutex

Correct Answer: Critical section

Explanation:

The critical section is the part of the code where the process/thread accesses shared variables or resources.

13 If a variable is incremented () by two threads concurrently without synchronization, what is the problem?

A. Starvation
B. Segmentation Fault
C. Lost Update
D. Deadlock

Correct Answer: Lost Update

Explanation:

Since incrementing is not atomic (Read-Modify-Write), one thread's update might overwrite the other's, leading to a lost update (Race Condition).

26 If a semaphore value is 0, what does sem_wait() do?

A. It destroys the semaphore.
B. It increments the value to 1.
C. It returns an error.
D. It blocks (waits) until the value becomes greater than 0.

Correct Answer: It blocks (waits) until the value becomes greater than 0.

Explanation:

sem_wait decrements the value. If the value is 0, it blocks until it can successfully decrement (i.e., when another thread posts).

27 What is the primary difference between pthread_mutex_lock and sem_wait on a binary semaphore?

A. There is no difference.
B. Semaphores are faster.
C. Mutexes can only be unlocked by the thread that locked them; Semaphores can be signaled by any thread.
D. Mutexes can be used between processes, semaphores cannot.

Correct Answer: Mutexes can only be unlocked by the thread that locked them; Semaphores can be signaled by any thread.

Explanation:

Mutexes imply ownership. Semaphores are signaling mechanisms and do not enforce ownership (any thread can sem_post).

Incorrect! Try again.

35 What is the danger of passing a pointer to a local variable (on the stack) to a thread created via pthread_create?

A. The creating function might return and deallocate the stack before the thread reads the variable.

D. It causes a compile error.

Correct Answer: The creating function might return and deallocate the stack before the thread reads the variable.

Local variables cease to exist when the function returns. If the thread runs after the parent function returns, it accesses invalid memory.

47 What is the concept of Priority Inversion?

This occurs when a lower-priority task holds a resource needed by a higher-priority task, effectively reducing the higher-priority task's execution speed.
