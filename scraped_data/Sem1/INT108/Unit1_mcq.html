1 Which of the following is a major difference between Python 2 and Python 3 regarding the print statement?

A. There is no difference
B. Python 3 uses parentheses for print, while Python 2 does not require them
C. Python 3 does not have a print function
D. Python 2 uses parentheses for print, while Python 3 does not require them

Correct Answer: Python 3 uses parentheses for print, while Python 2 does not require them

Explanation:

In Python 2, print is a statement (e.g., 'print 5'), whereas in Python 3, it is a function and requires parentheses (e.g., 'print(5)').
