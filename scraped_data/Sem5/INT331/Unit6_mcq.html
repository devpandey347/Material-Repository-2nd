1 What is the fundamental unit of work in Maven?

A. The Build Script
B. The Ant Task
C. The Project Object Model (POM)
D. The Gradle File

Correct Answer: The Project Object Model (POM)

Explanation:

Maven is based on the concept of a Project Object Model (POM). The POM contains information about the project and various configuration details used by Maven to build the project.

B. maven.xml
C. config.xml

Explanation:

The pom.xml file is the core of a project's configuration in Maven. It is a single configuration file that contains the majority of information required to build a project.

B. clean
C. site
D. debug

44 Which of the following describes the Dependency Mediation rule: 'Nearest Definition'?

B. Maven uses the version of the dependency declared deepest in the dependency tree
C. Maven errors out if duplicate dependencies are found
D. Maven uses the newest version available

Correct Answer: Maven uses the version of the dependency closest to your project in the dependency tree

Explanation:

If two versions of the same artifact are in the dependency tree, Maven picks the one 'nearest' to the project (fewest hops). If depths are equal, the first one declared wins.
