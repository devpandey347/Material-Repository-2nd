1 What is the primary goal of Enumeration in the ethical hacking process?

A. To physically access the server room
B. To encrypt the target's data
C. To gather specific information like user names, shares, and services from a system
D. To crash the target system

Correct Answer: To gather specific information like user names, shares, and services from a system

Explanation:

Enumeration involves active connections to a system to identify valid user accounts, shared resources, and service versions.

Incorrect! Try again.

2 Which phase of hacking immediately precedes Enumeration?

A. Maintaining Access
B. System Hacking
C. Scanning
D. Clearing Tracks

Correct Answer: Scanning

Explanation:

The standard hacking methodology order is Footprinting, Scanning, Enumeration, System Hacking, and then Maintaining Access.

Incorrect! Try again.

3 In Windows Enumeration, what is a 'Null Session'?

A. An unauthenticated connection to the IPC$ share
B. A session with administrator privileges
C. A session that has been timed out
D. A connection using an encrypted VPN

Correct Answer: An unauthenticated connection to the IPC$ share

Explanation:

A Null Session allows an anonymous user to connect to the Inter-Process Communication share (IPC$) with a null username and password.

Incorrect! Try again.

4 Which port is primarily associated with NetBIOS Name Service?

A. Port 443
B. Port 137
C. Port 21

Explanation:

NetBIOS Name Service typically uses UDP port 137.

nbtstat is a Windows utility specifically designed to troubleshoot and display NetBIOS over TCP/IP protocol statistics.

SNMP stands for Simple Network Management Protocol, used for managing devices on IP networks.

A. admin and user
B. public and private
C. default and secure
D. root and guest

Correct Answer: public and private

Explanation:

'public' is typically read-only, and 'private' is typically read-write. These defaults are often left unchanged, posing a security risk.

Explanation:

The Lightweight Directory Access Protocol (LDAP) is used to access and maintain distributed directory information services like Active Directory.

13 What is the definition of System Hacking?

A. Scanning a network for live hosts
B. Developing security policies
C. Monitoring network traffic for anomalies
D. The process of gaining access, escalating privileges, and hiding files

Correct Answer: The process of gaining access, escalating privileges, and hiding files

System hacking is the active phase of compromising a system, including cracking passwords, escalating privileges, and covering tracks.

14 Which password attack involves trying every possible combination of characters?

A. Social Engineering
B. Brute Force Attack
C. Dictionary Attack
D. Shoulder Surfing

Correct Answer: Brute Force Attack

Explanation:

A brute force attack attempts every possible alphanumeric combination until the correct password is found.

15 What is a 'Rainbow Table' used for?

A. Encrypting email communications
B. Looking up pre-computed password hashes
C. Visualizing network traffic
D. Storing firewall rules

Correct Answer: Looking up pre-computed password hashes

Explanation:

Rainbow tables are large databases of pre-computed hash chains used to reverse cryptographic hash functions, usually for cracking passwords.

Incorrect! Try again.

16 Adding random bits of data to a password before hashing it to defeat Rainbow Tables is called:

A. Salting
B. Spiceing
C. Masking
D. Peppering

Correct Answer: Salting

Explanation:

A salt is random data added to a password before hashing to ensure that the same password results in a different hash.

Incorrect! Try again.

A. In the boot.ini file
B. In the Registry SAM file

D. In the My Documents folder

Correct Answer: In the Registry SAM file

Explanation:

Local password hashes are stored in the Security Account Manager (SAM) database in the Windows Registry.

18 What is the legacy authentication protocol used by older Windows systems, known for vulnerabilities?

A. RADIUS

A. The Key Distribution Center (KDC)

21 What is 'Privilege Escalation'?

A. Moving laterally to another computer with same rights
B. Gaining higher-level access (e.g., Administrator) from a standard user account
C. Downgrading user rights to Guest
D. Resetting a password

Correct Answer: Gaining higher-level access (e.g., Administrator) from a standard user account

Explanation:

Privilege escalation is the act of exploiting a bug or configuration oversight to gain elevated access to resources that are normally protected.

24 What distinguishes a Computer Virus from a Worm?

A. There is no difference
B. A virus encrypts data, a worm deletes it
C. A virus requires a host program to replicate, while a worm is standalone
D. A virus travels over networks, a worm stays local

Correct Answer: A virus requires a host program to replicate, while a worm is standalone

Explanation:

A virus attaches itself to a host file/program to run. A worm is a standalone program that replicates itself to spread to other computers.

25 Which type of malware disguises itself as legitimate software to trick the user into installing it?

C. Trojan Horse

Correct Answer: Trojan Horse

Explanation:

Named after the Greek myth, a Trojan Horse appears useful or desirable but carries a malicious payload.

Correct Answer: To encrypt user files and demand payment for the decryption key

29 What is 'Scareware'?

A. Malware that tricks users into buying unnecessary software by claiming their computer is infected
B. A worm that spreads via email
C. A virus that deletes system 32
D. Software that screams when opened

Correct Answer: Malware that tricks users into buying unnecessary software by claiming their computer is infected

30 A type of malicious code that remains dormant until a specific event or date triggers it is called:

A. Logic Bomb
B. Adware
C. Spyware
D. Backdoor

Explanation:

A logic bomb is code inserted into software that executes a malicious function when specific criteria (like a date or employee termination) are met.

31 What is a 'Rootkit'?

A. Software designed to hide the existence of other malware and maintain privileged access
B. A database scanning tool
C. A password cracking tool
D. A kit for rooting Android phones

Correct Answer: Software designed to hide the existence of other malware and maintain privileged access

Explanation:

Rootkits modify the operating system to hide processes, files, and network connections, making detection very difficult.

32 A 'Polymorphic Virus' is difficult to detect because:

A. It changes its code or signature each time it infects a new file
B. It only runs on Linux
C. It is written in Python
D. It is invisible to the user

Correct Answer: It changes its code or signature each time it infects a new file

Explanation:

Polymorphic viruses encrypt their payload and use a different decryption routine for every infection to evade signature-based detection.

Incorrect! Try again.

33 What is a 'Macro Virus'?

A. A virus that attacks Mac computers
B. A virus written in the macro language of applications like Microsoft Word or Excel
C. A virus that infects the boot sector
D. A virus that is very large in file size

Correct Answer: A virus written in the macro language of applications like Microsoft Word or Excel

Explanation:

Macro viruses exploit the macro capabilities of office automation software to execute malicious code when a document is opened.

34 A network of compromised computers controlled by an attacker is called a:

A Botnet (Robot Network) consists of 'zombie' computers infected with malware that are controlled remotely by a botmaster.
