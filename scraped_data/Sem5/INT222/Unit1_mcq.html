1 Which JavaScript engine is Node.js built upon?

A. V8
B. Chakra
C. JavaScriptCore
D. SpiderMonkey

Node.js is built on Chrome's V8 JavaScript engine, which compiles JavaScript directly to native machine code.

A. Multi-threaded and non-blocking
B. Single-threaded and blocking
C. Multi-threaded and blocking
D. Single-threaded and non-blocking

Explanation:

Node.js uses a single-threaded event loop architecture but handles I/O operations asynchronously (non-blocking).

C. A blocking function

Explanation:

20 What is the convention for the first argument of a Node.js callback function?

A. The callback function itself
B. The result data
C. The configuration object
D. The error object (or null)

Explanation:

Node.js follows the 'error-first callback' convention, where the first argument is reserved for an error object (if any), and subsequent arguments are for success data.
