1 In Laravel, which method on the Illuminate\Http\Request instance is used to retrieve all input data as an array?

A. $request->get()
B. $request->input()
C. $request->obtain()
D. $request->all()

Correct Answer: $request->all()

Explanation:

The $request->all() method returns all of the incoming request's input data as an array.

Incorrect! Try again.

2 How can you retrieve a specific input value named 'username' from the request, returning 'Guest' if the value is not present?

A. $request->get('username', 'Guest')
B. $request->input('username', 'Guest')
C. $request->retrieve('username', 'Guest')
D. $request->username('Guest')

Correct Answer: $request->input('username', 'Guest')

Explanation:

The input method accepts a second argument which serves as the default value if the requested input key is not present.

Incorrect! Try again.
